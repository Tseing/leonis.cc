使用Pelican发布文章的流程

Leo | 2022年 7月12日

本文环境为Linux，首先在content目录中创建.nbdata与.ipynb的同名文件。.nbdata文件中保存了文章的结构信息，而.ipynb使用jupyter notebook保存了文章的具体内容。

In [ ]:
.nbdata
Title: 文章标题
Slug: 文章地址"://../<Slug>"
Date: 时间
Category: 类别
Tags: 标签
Author: 作者
Summary: 概要

在撰写文章后，进入虚拟环境，使用Pelican生成html文件。

In [ ]:
origin ./venv/bin/activate
pelican content -s publishconf.py

最后将output文件夹同步至Github中.github.io仓库即完成文章的发布。

In [ ]:
git add .
git commit
git push

在output文件夹中可以开启本地服务器，默认端口为8000，通过localhost:8000访问。

In [ ]:
python -m pelican.server
